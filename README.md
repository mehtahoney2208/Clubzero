# Colorful-Shop-Frontend-Vite

A colorful, frontend-only e-commerce demo built with React + Vite. Contains Add/Remove Cart, Search, Sort, Category filters, Dark Mode toggle, Product details modal, and a responsive Navbar + Footer.

## Run locally

1. Copy the project folder to your machine
2. Install dependencies

```bash
npm install
npm run dev
```

3. Open http://localhost:5173 in your browser
